# 小红书视频链接提取

Chrome 扩展，一键提取小红书笔记页面的视频链接，复制后可用下载工具下载。

## 安装

1. 下载或克隆本项目
2. 打开 Chrome，地址栏输入 `chrome://extensions/`
3. 右上角开启 **开发者模式**
4. 点击 **加载已解压的扩展程序**，选择 `xhs-video-downloader` 文件夹

## 使用

1. 打开小红书笔记页面（含视频的）
2. 点击浏览器工具栏的插件图标
3. 看到视频链接列表后：
   - **复制链接** — 粘贴到浏览器或下载工具中下载
   - **新标签页打开** — 浏览器直接播放，可右键另存

## 提取策略

- DOM `<video>` 标签检测
- 页面 SSR 数据解析
- Performance API 网络记录
- 网页源码正则匹配
- webRequest 网络请求拦截
